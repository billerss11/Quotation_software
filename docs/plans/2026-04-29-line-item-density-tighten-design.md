# Line Item Dense Typography Design

## Goal

Reduce the visible bulk of the line-item workbench without hiding fields or adding extra interactions.

## Direction

Use a denser typographic scale and tighter controls. Keep the same information architecture, but make parent cards, child rows, metric tiles, badges, and action buttons occupy less height.

## Scope

- Shrink line-item typography slightly.
- Reduce input heights and padding.
- Tighten child table columns and row spacing.
- Keep money columns visually strongest.
- Keep descriptions available, but less visually dominant.

## Constraints

- No calculation changes.
- No field removals in this pass.
- Preserve single-tax and mixed-tax behavior.
